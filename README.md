# SQL-project: Hiral Dasadia
The goal of this project is to design, build, and manage a comprehensive SQL-based relational database for a university. 
This project is demonstrating knowledge of:
•	Database design and entity relationships.
•	SQL queries for data retrieval, insertion, updates, and deletions.
•	Constraints, indexes, and query performance optimization.
•	Managing transactions and ensuring data integrity through normalization and referential integrity.
